Présentation du Projet : Prédiction de la Résiliation des Clients (Customer Churn)

L'objectif de ce projet est d'analyser et de prédire le départ des clients (« churn ») d'une entreprise de télécommunications à partir de leurs caractéristiques et comportements. En prédisant les clients susceptibles de partir, l'entreprise peut prendre des mesures pour améliorer leur fidélité.

Modélisation des variables

Nous avons utilisé la Régression Logistique pour prédire la résiliation des clients :

Entraînement du modèle sur les données prétraitées. Prédiction sur le jeu de test. Évaluation des performances avec la matrice de confusion, la précision, le rappel et la courbe ROC

from sklearn.model_selection import train_test_split
from sklearn.preprocessing import StandardScaler, LabelEncoder

# Remplacer les valeurs manquantes par la médiane
df['TotalCharges'] = df['TotalCharges'].fillna(df['TotalCharges'].median())


# Séparer les caractéristiques (X) et la cible (y)
X = df.drop(columns=['Churn'])
y = df['Churn']

# Encoder la variable cible (Yes -> 1, No -> 0)
label_encoder = LabelEncoder()
y = label_encoder.fit_transform(y)

# Identifier les variables catégoriques
categorical_features = X.select_dtypes(include=['object']).columns

# Appliquer l'encodage one-hot pour les variables catégoriques
X_encoded = pd.get_dummies(X, columns=categorical_features, drop_first=True)

# Standardiser les variables numériques
scaler = StandardScaler()
numerical_features = ['tenure', 'MonthlyCharges', 'TotalCharges']
X_encoded[numerical_features] = scaler.fit_transform(X_encoded[numerical_features])

# Diviser les données en ensemble d'entraînement et de test
X_train, X_test, y_train, y_test = train_test_split(X_encoded, y, test_size=0.2, random_state=42, stratify=y)

# Vérifier la forme des ensembles
X_train.shape, X_test.shape, y_train.shape, y_test.shape

import matplotlib.pyplot as plt
import seaborn as sns
from sklearn.metrics import roc_curve, auc

# Calcul des probabilités pour la courbe ROC
y_probs = model.predict_proba(X_test)[:, 1]  # Probabilité d'appartenance à la classe 1
fpr, tpr, _ = roc_curve(y_test, y_probs)
roc_auc = auc(fpr, tpr)

# Tracé de la courbe ROC
plt.figure(figsize=(8, 6))
plt.plot(fpr, tpr, color='blue', lw=2, label=f'ROC curve (AUC = {roc_auc:.2f})')
plt.plot([0, 1], [0, 1], color='gray', linestyle='--')  # Courbe aléatoire
plt.xlim([0.0, 1.0])
plt.ylim([0.0, 1.05])
plt.xlabel('Taux de faux positifs (FPR)')
plt.ylabel('Taux de vrais positifs (TPR)')
plt.title('Courbe ROC - Régression Logistique')

Recommandations

Pour réduire le churn, l'entreprise peut :

Offrir des réductions aux clients à risque.

Améliorer les services Internet et TV pour accroître la satisfaction.

Encourager des abonnements longue durée avec des offres attractives.
